# Octupus Collection

Collection o4n_dev_git helps developers to work with Gite Repositories.
By Ed Scrimaglia

## Required

Ansible >= 2.10  

## Modules

- o4n_git_acp  
  Add, Commit and Push

- o4n_git_set_remote  
  Set Git remote values on any directory to connect to central repository  

- o4n_git_import  
  import selected files from a Git repository
